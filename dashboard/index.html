How to run these tests

Netlify only serves this dashboard (HTML + JSON). It cannot start Playwright from your browser. Generate results.json on your machine or in GitHub Actions, then publish that file into dashboard/ for this site.

  • Local (Windows / Mac / Linux): START-HERE.md (simplest) · full RUNNING.md
  • GitHub (closest to “run from the web”): Actions → Run workflow — add repo secrets DREAM_NEIGHBORHOOD_EMAIL / DREAM_NEIGHBORHOOD_PASSWORD, download the explorer-results artifact, replace dashboard/results.json, commit, push.

This JSON reports no staging login env was recorded for the run. Many failures may be login_wall — that is an auth/session issue, not a property of the street address.

Smoke run size (1–1000)

This site does not run tests — it only remembers a number so you can copy the same value locally or in GitHub Actions. Optional: add ?count=2 to this page’s URL (combine with live mode as ?live=1&count=2).

Local command:

python text/test_neighborhood_explorer.py --count 2

In GitHub → Actions → Run workflow, set the count input to the same number.
